# Resume

A simple resume written in React. See it in action [here](https://ClaytonPassmore.github.io/resume).

## Installation

Assuming you have npm, it's as easy as this:

```shell
$ npm install
```

## Running

Start the server:

```shell
$ npm start
```

## Deploying

Use the GitHub pages package to deploy:

```shell
$ npm run deploy
```
